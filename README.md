# speechRecognitionDemo

语音识别的小demo

## 功能
1. 调用讯飞平台进行语音识别
2. 调用百度平台对语音提到的地点进行地图打印
3. 调用天气API对语音提到的提点进行天气显示

## 注意
1. 注释中给出各平台的官方文档
2. speechRecognitionDemo为主程序
